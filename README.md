# Silent Struggle Dashboard

## Background
Solusi ini dirancang untuk membantu Smart Campus UB memahami support gap kesehatan mental mahasiswa berbasis data.

## Problem
Banyak mahasiswa memiliki indikasi mental health, tetapi belum mencari bantuan profesional.

## Dataset
Student Mental Health Dataset.

## Method
Data cleaning, feature engineering, support gap segmentation, exploratory data analysis, visualization, dan dashboard recommendation design.

## Key Features
- Support status segmentation
- Silent struggle analysis
- Year study analysis
- CGPA analysis
- Co-occurrence mental health indicators
- Smart campus recommendation

## Limitation
Dataset kecil, self-reported, bukan alat diagnosis medis.

## Output
Dashboard concept, visualization, insight, dan rekomendasi implementasi.
